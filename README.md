# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (Ã  la bit.ly). Users can keep track of the total number of views a link they have created has received, as well as when it was created.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the 'npm install' command).
- Run the development web server using the 'node express_server.js' command.
